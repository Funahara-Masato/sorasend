# sorasend 🌙

**「今夜の空を、あなたのいる夜に届ける」**

撮影した夜空の写真を、遠く離れた場所にいる大切な人の"夜"に合わせて届けるWebアプリです。
日本との時差を計算し、相手の地域で夜になるタイミングまでカウントダウンします。

## 使い方

1. 届けたい相手の地域を選択（ベトナム・タイ・シンガポール・中国・アメリカ・イギリス・ドイツなど）
2. 夜空の写真をアップロード、またはキャンバスで夜空を描く
3. メッセージを添えて「送る」
4. 相手の地域で夜になるまでのカウントダウンを待つ
5. カウントダウンが0になると、相手への配信画面が表示される

## 技術スタック

- HTML / CSS / JavaScript（Vanilla）
- タイムゾーン計算（各国との時差処理）
- Canvas API（夜空のアニメーション生成）
- 複数スクリーン遷移の状態管理

## 背景

幼少期をベトナム（ホーチミン）で過ごした経験から、時差のある場所にいる人と夜空を共有したいという思いで作りました。「同じ夜空でも、届く時間が違う」というコンセプトを形にしたプロジェクトです。

## 実行方法

`index.html` をブラウザで開くだけで動作します（サーバー不要）。

```bash
open index.html
```

## ファイル構成

```
sorasend/
├── index.html   # UI・スクリーン構成
├── script.js    # タイムゾーン計算・カウントダウン・Canvas描画
└── style.css    # スタイル定義
```
